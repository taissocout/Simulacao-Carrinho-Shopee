# 🇧🇷 Português

## 📌 Sobre o Projeto

O **Shopee Cart Simulator** é uma aplicação CLI construída em Node.js que simula uma experiência real de compra em um e-commerce.

✔️ Experiência interativa no terminal  
✔️ Lógica real de carrinho e checkout  
✔️ Simulação de pagamento com desconto  
✔️ Feedback visual com animações  

---

## 🚀 Funcionalidades

### 🛒 Carrinho
- Adicionar produtos com quantidade  
- Remover item completo ou parcial  
- Visualizar carrinho atualizado  
- Cálculo automático de total  

### 💳 Checkout
- Simulação de pagamento  
- Descontos:
  - Pix → 10%
  - Cartão → 5%
  - Boleto → 8%  
- Código de rastreio  
- Limpeza do carrinho após compra

# 🛍️ Shopee Cart Simulator (Interactive CLI)

---

# 🇧🇷 Português

## 📌 Sobre o Projeto

O **Shopee Cart Simulator** é uma aplicação CLI desenvolvida em Node.js que simula uma experiência real de e-commerce.

- ES Modules  
- Async/Await  
- CLI UX  
- Lógica de carrinho e pagamento  

---

## 🚀 Funcionalidades

- Adicionar/remover produtos  
- Visualizar carrinho  
- Checkout com desconto  
- Código de rastreio  
- Menu interativo  

---

## ⚙️ Execução

git clone https://github.com/taissocout/shopee-cart-simulator.git  
cd shopee-cart-simulator  
node main.js
